What is Web Scraping?
Its used to identify and download text from one or more pages or
archives within a single website (Ignatow and Mihalcea 2017:82).
The user writes a scriptin our case, an R scriptthat species
where on the site to look for data and what type of data to scrape
and collect.
Scraped data are then usually put in matrix or data frame form for
later analysis.
Useful when you need to collect a lot of data (imagine doing this
manually)!
Users can create their own scrapers, but for this class, we will focus
on using pre-made scrapers. Today well learn how to scrape Twitter.

Using an API
All websites are stored on servers.
When you type in the URL for a website or webpage, your web
browser sends out a request to the appropriate server.
The server will then respond to your browserhopefully in the form
of the website or webpage that you requested.
The portion of the server responsible for processing requests and
making responses is the application programming interface, or
API (Gazarov 2016).
Think of an API as a sort of user interface, but instead of making
navigation easier for the human eye, it makes navigation easier for
client software (i.e., software on your data-requesting computer)
(Berlind 2015).

Twitters REST API
Today we will learn how to scrape Twitter data through interacting
with the REST (Representational State Transfer) API (Carey 2009).

published over the past 7 days (Twitter Developer 2019).
Anything older than 7 dayswhich we called historical Twitter

ourselves with Twitters API.
Go to apps.twitter.com and sign in using your Twitter login credentials. If
you dont have a Twitter account, you can either (a) quickly create one, or

Plug in your app name and the numbers into the following code and run it
