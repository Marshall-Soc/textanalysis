Introduction
In addition to named-entity recognition, another popular tool under
the umbrella of NLP is part-of-speech (POS) tagging.
POS taggers classify each word in each document by its part of
speech: e.g., whether a word is a noun, verb, adverb, preposition, and
so on.
Instructor: Marshall A. Taylor SOC/MDSC 43919 February 13, 2019 2 / 30
How does POS Tagging Work?
Like the NER classier we used on Monday, one of the POS taggers
we will be is part of the OpenNLP library (The Apache Software
Foundation 2017).
As the tagger documentation says: The Part of Speech Tagger
marks tokens with their corresponding word type based on the token
itself and the context of the token. A token might have multiple pos
tags depending on the token and the context. The OpenNLP POS
Tagger uses a probability model to predict the correct pos tag out of
the tag set (The Apache Software Foundation 2019).
