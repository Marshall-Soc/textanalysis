What it is
An introduction to basic, intermediate, and (some) advanced forms of
computational text analysis frequently used in social science
applications.
A hands-on course designed to (1) give you some foundational skills
in computational text analysis, and (2) recognize when and where to
apply these skills in social science research projects.
An elective in the Data Science minor (MDSC).
A conceptually-oriented course.
Instructor: Marshall A. Taylor SOC/MDSC 43919 January 16, 2019 2 / 16
What it is not
A course that assumes any background knowledge in programming
(though you can certainly benet from this prior exposure).
A course that assumes any background knowledge in statistics
(though, again, it will help).
A deep-dive into the mathematics or algorithms behind the methods
well be talking about.
A computer science course.

The Game Plan
Objectives: By the end of this course, you should be able to:
interpret, question, and discuss text analysis methods accurately and
with an understanding of which type of method should be used in
particular cases.
use basic programming and data analysis skills in the R statistical
computing environment.
